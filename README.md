# GPS-da-area-particular-silvia
🌍📍 GPS da Área Particular SilvaO projeto GPS da Área Particular Silva foi desenvolvido com o objetivo de facilitar a locomoção e promover mais autonomia para pessoas com deficiência visual dentro do ambiente escolar. Utilizando tecnologia acessível, o sistema funciona por comando de voz, ajudando o usuário a encontrar locais
